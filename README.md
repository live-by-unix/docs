# Why is localhost NOT reccomended?       
It has not been tested, but localhsot might break the collab feature and cause longer boot times.    
This is because Cloudflare has servers all around the world. This creates it so that every app hosted is fast.      
So, imagine localhost being one private, unstable, and not so performance good sever. For another perosn to visit the website, you will need to do port forwarding.    
Port forwarding exposes your network anyway, BUT EVEN LIKE that the website will be slow clunky, and most collab will also break.       
That is why localhost is not reccomended, but you can use it to test out different versions.
